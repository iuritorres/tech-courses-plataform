Nesse curso você vai entender na prática, como é trabalhar com Automação de Processos e pipipi popopo…

exemplo de Robot Process Automation com Python:

```python
def __request_html(page_url: str) -> Firefox:
    options = FirefoxOptions()
    options.add_argument('--headless')
    options.add_argument('--private')

    browser = Firefox(options=options)
    browser.get(page_url)

    # Roll to the end (load all elements)
    browser.find_element(By.TAG_NAME, 'html').send_keys(Keys.END)
    sleep(5)

    return browser
```
